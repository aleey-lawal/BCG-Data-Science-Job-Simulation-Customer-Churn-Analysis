# BCG-Data-Science-Job-Simulation-Customer-Churn-Analysis
This repository contains my work from the Boston Consulting Group (BCG) Data Science Virtual Experience Program completed on the Forage platform. The simulation provided a hands-on opportunity to experience real-world data science workflows within a consulting environment.
Project Overview

In this project, I developed a customer churn analysis solution using Python, simulating how a data science team at BCG might approach a business problem. The objective was to identify key drivers of customer churn and provide actionable insights to support data-driven decision-making.

🧠 Key Skills Demonstrated
Data cleaning and preprocessing of real-world datasets
Exploratory Data Analysis (EDA) to uncover patterns and trends
Feature engineering to improve model performance
Predictive modeling for churn classification
Translating analytical findings into business insights
Communicating results through an executive summary
🛠️ Tools & Technologies
Python (Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn)
Jupyter Notebook
Data visualization techniques
Business storytelling
📈 Key Outcomes
Identified critical factors influencing customer churn
Built a predictive model to classify at-risk customers
Delivered insights in a clear, business-focused format
Strengthened ability to bridge technical analysis and stakeholder communication
📄 Executive Summary

As part of the simulation, I prepared an executive-level summary highlighting:

Key churn drivers
Model performance and reliability
Strategic recommendations to reduce churn
